# 水印工具 PWA

一个可以在浏览器中运行的离线水印工具，支持批量处理文件夹中的所有文件。

## 功能特点

- **拖拽上传** - 直接拖拽文件或文件夹到页面
- **批量处理** - 一次性处理整个文件夹
- **多种格式** - 支持 PDF、Word、Excel、图片
- **自定义水印** - 可设置文字、颜色、透明度、位置
- **复制模式** - 生成带水印的新文件，不修改原文件
- **ZIP 打包** - 处理完成后一键下载所有文件
- **PWA 支持** - 可安装为桌面应用，离线可用
- **跨平台** - Mac、Windows、Linux 都可以使用

## 使用方法

### 在线使用

直接用浏览器打开 `index.html` 文件即可使用。

### 本地运行

```bash
# 使用 Python 启动简单服务器
cd pwa-watermark
python3 -m http.server 8000

# 或使用 Node.js
npx serve .

# 然后打开浏览器访问
# http://localhost:8000
```

### 安装为桌面应用

1. 用 Chrome 或 Edge 打开应用
2. 点击地址栏右侧的安装图标
3. 选择安装应用

## 支持的文件格式

| 格式 | 扩展名 | 支持状态 |
|------|--------|----------|
| PDF  | .pdf   | ✅ 完全支持 |
| Word | .docx  | 🔄 基础支持 |
| Excel| .xlsx  | 🔄 基础支持 |
| 图片 | .png, .jpg, .jpeg | ✅ 完全支持 |

## 水印选项

| 选项 | 说明 |
|------|------|
| 文字 | 自定义水印文字 |
| 颜色 | 选择水印颜色 |
| 透明度 | 10% - 100% |
| 位置 | 居中 / 平铺 / 对角线 |

## 技术栈

- 纯前端实现，无需后端
- PDF-lib - PDF 处理
- JSZip - ZIP 打包下载
- Service Worker - 离线支持

## 安全说明

- 所有文件处理都在本地浏览器完成
- 不会上传任何文件到服务器
- 原文件不会被修改
